## js简答题

1. js中的单行和多行注释如何表示?

   单行注释为   // 注释

   多行注释为   `/*注释*/`

2. 什么是变量,声明变量有哪几种方式？

   值可以改变的量，用来保存数据时使用

   声明变量的两种方式：

   1. 定义变量不赋值

   2. 定义变量并赋值

   3. 同时定义多个变量

      `var 变量名1, 变量名2`

   4. 同时定义多个变量并赋值

   

3. 变量名命名可以包含什么？其中不能以什么开头？

   变量命名可以包含数字，下划线，`$`，大小写字母

   不能以数字开头

4. 变量名命名为什么区分大小写？不能拿什么用来命名？

   大小写不同，相当于不同的变量。

   不能用关键字命名，而且不推荐使用关键字
